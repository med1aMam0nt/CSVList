Цель: считать данные о людях из CSV-файла и получить объект типа List<Person>, используя встроенные коллекции Java.

Для подразделений создается отдельная сущность Department, а ее ID генерируется в программе.
Из исходника создается также сущность Person с полями:
id — ID человека (берётся из CSV)
name — имя
gender — пол
department — объект Department
salary — зарплата
birthDate — дата рождения

Разделитель: 
;

Формат даты рождения: 
дд.мм.гггг

Порядок колонок:
id;name;gender;department;salary;birthDate

Пример строки:
1;Ivan Ivanov;Male;IT;120000;21.05.1999

Дополнительно: если первая строка выглядит как заголовок — она пропускается автоматически, а подразделения переиспользуются через Map<String, Department>, чтобы одинаковые названия не создавались лишний раз
Файл CSV должен лежать в корне рабочей папки проекта
